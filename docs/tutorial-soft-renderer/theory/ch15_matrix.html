PART 4 · CHAPTER 15
矩陣與線性變換
縮放與旋轉的統一語言——用一個矩陣描述整個空間的變形方式
matrix 線性變換 基向量 旋轉矩陣 複合變換 正交矩陣

Ch13 和 Ch14 建立了向量的工具：點積衡量方向一致性，叉積計算法向量。 但要把一個 3D 物件「縮放」「旋轉」「放到世界裡」，需要一個更系統化的框架—— 矩陣（Matrix）。

矩陣不只是數字排列的表格；它精確描述了一個線性變換（linear transform）， 也就是「空間如何被拉伸、旋轉、剪切」。理解矩陣的幾何意義， 比背公式更重要。

1
矩陣乘法：列點積欄

結果矩陣 \(C = AB\) 中，第 \(i\) 列、第 \(j\) 欄的值，等於 \(A\) 的第 \(i\) 列與 \(B\) 的第 \(j\) 欄的點積：

$C_{ij} = \sum_{k} A_{ik} \cdot B_{kj}$

具體以 2×2 為例：

$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} e & f \\ g & h \end{pmatrix} = \begin{pmatrix} ae+bg & af+bh \\ ce+dg & cf+dh \end{pmatrix}$

矩陣乘以向量（向量視為 \(n \times 1\) 的行矩陣）：

$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}$

要使 \(AB\) 合法，\(A\) 的欄數必須等於 \(B\) 的列數。 3D 圖形裡最常見的情形是 \(3 \times 3\) 矩陣乘以 \(3 \times 1\) 向量，或 \(3 \times 3\) 乘以 \(3 \times 3\)。

矩陣乘法不滿足交換律：\(AB \neq BA\)（一般情形）
這個性質在複合變換中至關重要——先旋轉再縮放，和先縮放再旋轉，結果完全不同。 後面的第 5 節會用具體數字示範。

2
線性變換的幾何意義：基向量去了哪裡

一個 \(3 \times 3\) 矩陣 \(M\) 的每一欄就是三個標準基向量 \(\mathbf{e}_1, \mathbf{e}_2, \mathbf{e}_3\) 經過變換後的新位置：

$M = \begin{pmatrix} \textcolor{#89b4fa}{m_{00}} & \textcolor{#a6e3a1}{m_{01}} & \textcolor{#94e2d5}{m_{02}} \\ \textcolor{#89b4fa}{m_{10}} & \textcolor{#a6e3a1}{m_{11}} & \textcolor{#94e2d5}{m_{12}} \\ \textcolor{#89b4fa}{m_{20}} & \textcolor{#a6e3a1}{m_{21}} & \textcolor{#94e2d5}{m_{22}} \end{pmatrix}$

第一欄（藍）\(= M\mathbf{e}_1\)，第二欄（綠）\(= M\mathbf{e}_2\)，第三欄（青）\(= M\mathbf{e}_3\)

因此，任意向量 \(\mathbf{v} = (v_x, v_y, v_z)\) 的變換結果可以理解為：

$M\mathbf{v} = v_x \cdot \textcolor{#89b4fa}{\text{（第一欄）}} + v_y \cdot \textcolor{#a6e3a1}{\text{（第二欄）}} + v_z \cdot \textcolor{#94e2d5}{\text{（第三欄）}}$

3
縮放矩陣

縮放只改變各基向量的長度而不改變方向，因此縮放矩陣是對角矩陣：

$S(s_x, s_y, s_z) = \begin{pmatrix} s_x & 0 & 0 \\ 0 & s_y & 0 \\ 0 & 0 & s_z \end{pmatrix}$

驗證——把標準基向量代入：

$S\,\mathbf{e}_1 = (s_x,\, 0,\, 0), \quad S\,\mathbf{e}_2 = (0,\, s_y,\, 0), \quad S\,\mathbf{e}_3 = (0,\, 0,\, s_z)$

均勻縮放（uniform scale，\(s_x = s_y = s_z = k\)）等價於 \(k\mathbf{I}\)，其中 \(\mathbf{I}\) 是單位矩陣。 縮放的逆矩陣只需取各分量倒數：\(S(s_x, s_y, s_z)^{-1} = S(1/s_x, 1/s_y, 1/s_z)\)。

4
旋轉矩陣推導

從 2D 出發

在 2D 平面中，逆時針旋轉角度 \(\theta\) 的矩陣，直接從旋轉後的基向量讀出： \(\mathbf{e}_1 = (1,0)\) 旋轉後到 \((\cos\theta,\, \sin\theta)\)， \(\mathbf{e}_2 = (0,1)\) 旋轉後到 \((-\sin\theta,\, \cos\theta)\)。 把這兩個向量排成矩陣的各欄：

$R(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$

3D 旋轉矩陣

3D 旋轉繞固定軸進行，旋轉時該軸方向的分量不變，另外兩個分量按 2D \(R(\theta)\) 旋轉。 三個軸的旋轉矩陣（逆時針，右手定則）：

$R_x(\theta) = \begin{pmatrix} 1 & 0 & 0 \\ 0 & \cos\theta & -\sin\theta \\ 0 & \sin\theta & \cos\theta \end{pmatrix}, \quad R_x \text{ 讓 } x \text{ 軸固定不動}$ $R_y(\theta) = \begin{pmatrix} \cos\theta & 0 & \sin\theta \\ 0 & 1 & 0 \\ -\sin\theta & 0 & \cos\theta \end{pmatrix}, \quad R_y \text{ 讓 } y \text{ 軸固定不動}$ $R_z(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta & 0 \\ \sin\theta & \cos\theta & 0 \\ 0 & 0 & 1 \end{pmatrix}, \quad R_z \text{ 讓 } z \text{ 軸固定不動}$
\(R_y\) 的符號為何和 \(R_x\)、\(R_z\) 不同？
繞 \(y\) 軸旋轉時，另外兩個作用軸是 \(z\) 和 \(x\)（循環 \(x \to z\)，方向相反於 \(x \to y\)）。 右手定則下，\(R_y\) 中 \(\sin\theta\) 的位置相對 \(R_x\)、\(R_z\) 而言是「翻轉」的。 最可靠的推導方式：永遠從「旋轉後的基向量位置」讀出矩陣各欄，不要靠記憶符號。

5
複合變換：乘法順序的幾何意義

要對一個向量先做旋轉 \(R\)，再做縮放 \(S\)：

$\mathbf{v}' = S\,(R\,\mathbf{v}) = (SR)\,\mathbf{v}$

因為矩陣乘法滿足結合律，可以先把 \(SR\) 算成一個矩陣，再乘以 \(\mathbf{v}\)。 這就是 GPU 能把整個 MVP 矩陣預先算好、每個頂點只做一次乘法的原因。

\(SR \neq RS\)（一般情形）——矩陣右邊的先作用，左邊的後作用。
寫 \(M = SR\) 意思是「先 \(R\) 後 \(S\)」。先縮放再旋轉 vs 先旋轉再縮放，對不均勻縮放（如 \(s_x \neq s_y\)）結果截然不同。

具體數字示範，\(S =\) 沿 \(x\) 縮放 2 倍，\(R =\) 旋轉 90°：

$SR = \begin{pmatrix}2&0\\0&1\end{pmatrix}\begin{pmatrix}0&-1\\1&0\end{pmatrix} = \begin{pmatrix}0&-2\\1&0\end{pmatrix}$ $RS = \begin{pmatrix}0&-1\\1&0\end{pmatrix}\begin{pmatrix}2&0\\0&1\end{pmatrix} = \begin{pmatrix}0&-1\\2&0\end{pmatrix}$

6
正交矩陣：旋轉矩陣的特殊性質

旋轉不改變向量的長度和夾角，只改變方向。 這讓旋轉矩陣成為正交矩陣（orthogonal matrix），具備一個極重要的性質：

$\boxed{M^{-1} = M^\top} \quad \Longleftrightarrow \quad MM^\top = I$

對 2D \(R(\theta)\) 直接驗證：

$R(\theta)\,R(\theta)^\top = \begin{pmatrix}\cos\theta & -\sin\theta\\\sin\theta & \cos\theta\end{pmatrix} \begin{pmatrix}\cos\theta & \sin\theta\\-\sin\theta & \cos\theta\end{pmatrix} = \begin{pmatrix}1&0\\0&1\end{pmatrix} = I \checkmark$
法向量變換的陷阱：在 3D 渲染中，法向量從物件空間變換到世界空間時， 不能直接乘以 Model 矩陣 \(M\)——必須乘以 \((M^{-1})^\top\)（逆轉置矩陣）。

若 \(M\) 只包含旋轉（正交矩陣），則 \((M^{-1})^\top = (M^\top)^\top = M\)， 法向量和位置用同一個矩陣即可。
若 \(M\) 包含不均勻縮放（\(s_x \neq s_y \neq s_z\)）， 逆轉置與 \(M\) 不同，必須分開計算。
矩陣類型逆矩陣計算成本
旋轉矩陣（正交，\(\det = 1\)） \(R^{-1} = R^\top\) \(O(n^2)\) 轉置
縮放矩陣（對角） \(S(s_x,s_y,s_z)^{-1} = S(1/s_x, 1/s_y, 1/s_z)\) \(O(n)\) 取倒數
一般可逆矩陣 Gauss-Jordan 消去 \(O(n^3)\)
奇異矩陣（\(\det = 0\)） 不可逆（空間被壓扁到更低維度） —

C++ 實作

// 3×3 矩陣，row-major 儲存：m[row][col] struct Mat3 { float m[3][3]; // 預設建構：單位矩陣 Mat3() { for (int i = 0; i < 3; i++) for (int j = 0; j < 3; j++) m[i][j] = (i == j) ? 1.f : 0.f; } // 矩陣 × 矩陣：C_ij = Σ_k A_ik * B_kj Mat3 operator*(const Mat3& b) const { Mat3 r; for (int i = 0; i < 3; i++) for (int j = 0; j < 3; j++) { r.m[i][j] = 0; for (int k = 0; k < 3; k++) r.m[i][j] += m[i][k] * b.m[k][j]; } return r; } // 矩陣 × 向量（v 視為 3×1 行向量） Vec3 operator*(Vec3 v) const { return { m[0][0]*v.x + m[0][1]*v.y + m[0][2]*v.z, m[1][0]*v.x + m[1][1]*v.y + m[1][2]*v.z, m[2][0]*v.x + m[2][1]*v.y + m[2][2]*v.z }; } // 轉置（對正交矩陣即逆矩陣） Mat3 transpose() const { Mat3 r; for (int i = 0; i < 3; i++) for (int j = 0; j < 3; j++) r.m[i][j] = m[j][i]; return r; } }; // 縮放矩陣 Mat3 scale3(float sx, float sy, float sz) { Mat3 m; m.m[0][0] = sx; m.m[1][1] = sy; m.m[2][2] = sz; return m; } // 繞 Z 軸旋轉（弧度，逆時針） Mat3 rotateZ(float theta) { float c = std::cosf(theta), s = std::sinf(theta); Mat3 m; m.m[0][0]=c; m.m[0][1]=-s; m.m[1][0]=s; m.m[1][1]= c; return m; } // 繞 Y 軸旋轉（注意符號：sin 在 [0][2] 而非 [2][0]） Mat3 rotateY(float theta) { float c = std::cosf(theta), s = std::sinf(theta); Mat3 m; m.m[0][0]= c; m.m[0][2]= s; m.m[2][0]=-s; m.m[2][2]= c; return m; } // 先旋轉再縮放（S·R）：rotateY 先作用，scale3 後作用 Mat3 M = scale3(2, 1, 1) * rotateY(0.5f);
Row-major vs Column-major：本章 Mat3 採 row-major 儲存（m[row][col]）， 乘法寫作 M * v（矩陣在左、列向量在右）。
OpenGL GLSL 使用 column-major，同樣的變換在 GLSL 中記憶體排列不同，但數學結果相同。 實務上切換時只需注意：column-major 矩陣的 C++ 記憶體排列等同於 row-major 的轉置。

小結

✓ 本章掌握的概念
  • 矩陣乘法：\(C_{ij} = \sum_k A_{ik} B_{kj}\)，不交換
  • 線性變換：矩陣各欄 = 基向量的新位置
  • 縮放矩陣：對角矩陣，逆為取倒數
  • 旋轉矩陣：從旋轉後基向量位置推導
  • 複合變換：\(SR\mathbf{v}\)，右邊先作用
  • 正交矩陣：\(R^{-1} = R^\top\)
