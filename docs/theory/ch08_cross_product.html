PART 3 · CHAPTER 08
叉積與有向面積
三角形填色的數學基礎 — 判斷「點在線段的哪一側」
cross product signed area determinant 向量

你已經知道怎麼畫一條線了。接下來的目標是填滿一個三角形。

問題聽起來很直接——三角形有三個頂點，把裡面的像素全部點亮。 但關鍵問題在於：給定一個像素座標 (x, y)，怎麼判斷它在三角形裡面還是外面？

這一章建立核心工具：2D 叉積（cross product）。
它能回答一個更基礎的問題：點 P 在線段 AB 的左邊還是右邊？
先搞懂這一件事，三角形內外測試就會自然推導出來。

1
先退一步：只考慮一條邊

想像你站在 A 點，面向 B 點走去。點 P 在你的左手邊，還是右手邊？

這個「左右判斷」就是三角形填色的核心。 一個三角形有三條邊，如果點 P 對所有三條邊都在「正確那一側」， 它就在三角形內。

現在的問題是：怎麼計算「哪一側」？

2
2D 叉積的定義

先回顧一下 3D 叉積。兩個 3D 向量 u 和 v 的叉積是：

u × v = (uyvz − uzvy, uzvx − uxvz, uxvy − uyvx)

現在把場景壓平到 2D，令 z = 0：

u = (ux, uy, 0) v = (vx, vy, 0) u × v = (0, 0, uxvy − uyvx) ↑ 只剩 z 分量不為零

在 2D 中，叉積退化成一個純量（scalar）——只有 z 分量：

cross(u, v) = ux · vy − uy · vx

這個數值就是 2D 叉積。兩行，一個減法，就夠了。

套入「點在線段哪一側」

給定線段 AB 和點 P，令：

AB = B − A = (Bx−Ax, By−Ay) AP = P − A = (Px−Ax, Py−Ay)

則：

cross(AB, AP) = ABx · APy − ABy · APx 展開： = (Bx−Ax)(Py−Ay) − (By−Ay)(Px−Ax)

結果是一個數字，正的或負的（或零，代表 P 剛好在直線上）。 但這個數字的意義是什麼？

3
幾何意義：有向面積

叉積的幾何意義是：兩個向量張開的平行四邊形的「有向面積」。

更精確地說：

cross(u, v) = |u| · |v| · sin(θ) θ = 從 u 到 v 的有向角 |cross(u, v)| = 底 × 高 = 平行四邊形面積

「有向」的意思是：面積帶有正負號，取決於 v 是在 u 的逆時針方向還是順時針方向：

▌ cross > 0

v 在 u 的逆時針方向（左側）
P 在 AB 的左邊

在數學座標系（y 朝上）成立。
若 y 朝下（螢幕座標），符號相反——
但一致性才是重點，只要全程用同一套即可。

▌ cross < 0

v 在 u 的順時針方向（右側）
P 在 AB 的右邊

重點：你不需要背「左邊是正還是負」。
只需要記得：三條邊用同樣的順序（都順時針或都逆時針）計算， 三個結果同號 → 點在三角形內。

4
行列式形式

叉積有一個很常見的等價寫法——2×2 行列式：

cross(u, v) = det ⎡ ux vx ⎤ ⎣ uy vy ⎦ = ux · vy − uy · vx

展開後和前面的定義完全相同。行列式形式讓你更容易記住—— 對角相乘再相減：左上×右下 − 左下×右上。

用三個頂點直接寫出來

如果你有三角形頂點 A、B、C（不是向量，是點座標）， 可以直接把它們代入：

AB = (Bx−Ax, By−Ay) AP = (Px−Ax, Py−Ay) cross(AB, AP) = (Bx−Ax)(Py−Ay) − (By−Ay)(Px−Ax)

這六個數字（Ax、Ay、Bx、By、Px、Py）全部是整數座標， 結果也是整數——不需要浮點數。這在光柵化實作中很有用。

5
三角形的有向面積

叉積還帶給你另一個禮物：三角形面積公式。

平行四邊形面積 = |cross(AB, AC)|
三角形面積 = 一半：

Area(△ABC) = ½ · |cross(AB, AC)| 展開： = ½ · |(Bx−Ax)(Cy−Ay) − (By−Ay)(Cx−Ax)|

去掉絕對值（允許負值），就得到有向面積。 它的正負號告訴你：三個頂點是逆時針排列（正）還是順時針排列（負）：

SignedArea(△ABC) = ½ · cross(AB, AC) > 0 : A→B→C 逆時針（數學座標系） < 0 : A→B→C 順時針 = 0 : 三點共線（退化三角形）
為什麼有向面積很重要？
下一章的邊緣函數（Edge Function）計算的值， 可以除以整個三角形的有向面積，得到重心座標（第 Ch10 章的主題）。
這就是為什麼三角形內外測試和屬性插值（顏色、UV、法線…） 可以用同一套計算框架完成。

6
用程式碼表達

把上面的數學翻譯成 C++：

// 2D 叉積（純量）
// 給定兩個 2D 向量 u、v，回傳 u.x*v.y - u.y*v.x
float cross2d(Vec2 u, Vec2 v) {
    return u.x * v.y - u.y * v.x;
}

// 點 P 在線段 AB 的哪一側？
// 結果 > 0 → 左側；< 0 → 右側；= 0 → 在直線上
float edgeTest(Vec2 A, Vec2 B, Vec2 P) {
    Vec2 AB = { B.x - A.x, B.y - A.y };
    Vec2 AP = { P.x - A.x, P.y - A.y };
    return cross2d(AB, AP);
}

// 三角形有向面積（面積 = 回傳值的一半）
float signedArea(Vec2 A, Vec2 B, Vec2 C) {
    return edgeTest(A, B, C); // cross(AB, AC)
}

就這樣。edgeTest() 是後面所有東西的核心—— 只有一個乘法、一個減法、結果帶正負號。

7
本章小結

本章學到的三件事：

1. 2D 叉積 = u.x·v.y − u.y·v.x，是一個純量，不是向量。
2. 它的絕對值是兩向量張開的平行四邊形面積（三角形面積的兩倍）。
3. 它的正負號告訴你 v 在 u 的哪一側——這就是判斷「點在線段左/右」的依據。

下一章：我們用三次 edgeTest()（對三條邊各做一次）， 建立完整的邊緣函數（Edge Function）， 把「點在三角形內嗎？」這個問題完整解決。
