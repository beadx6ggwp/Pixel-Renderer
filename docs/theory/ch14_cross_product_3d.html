PART 4 · CHAPTER 14
3D 叉積（Cross Product）
從三角形的兩條邊計算法向量——N·L 光照裡那個 N 就是這樣來的
cross product 法向量 右手定則 sinθ 反交換律 背面剔除

Ch13 的 N·L 光照公式裡，N 是表面法向量（surface normal）——一個垂直於三角形平面的單位向量。 但我們從來沒說過它是如何計算出來的。答案是叉積（Cross Product）。

叉積接受兩個 3D 向量，回傳第三個向量，且這個結果向量垂直於前兩者所構成的平面。 只要取三角形的兩條邊，叉積就能給出那個三角形的法向量。

1
定義：行列式展開公式

給定兩個 3D 向量 a = (ax, ay, az) 和 b = (bx, by, bz)， 它們的叉積 a × b 定義為下列 3×3 行列式的展開：

$\mathbf{a} \times \mathbf{b} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ a_x & a_y & a_z \\ b_x & b_y & b_z \end{vmatrix}$

沿第一列展開（cofactor expansion）：

$= \textcolor{#89b4fa}{\mathbf{i}}\,(a_y b_z - a_z b_y) - \textcolor{#89b4fa}{\mathbf{j}}\,(a_x b_z - a_z b_x) + \textcolor{#89b4fa}{\mathbf{k}}\,(a_x b_y - a_y b_x)$

整理為分量形式（實作只需記這一行）：

$\mathbf{a} \times \mathbf{b} = \bigl(\;\textcolor{#a6e3a1}{a_y b_z - a_z b_y},\;\;\textcolor{#a6e3a1}{a_z b_x - a_x b_z},\;\;\textcolor{#a6e3a1}{a_x b_y - a_y b_x}\;\bigr)$

其中 i, j, k 是 x, y, z 方向的單位基向量（standard basis vectors）。 行列式展開是推導公式的方法，但實際寫程式只需要記住最後那行分量形式。

記憶法：循環置換（cyclic permutation）
x 分量：y·z − z·y（跳過 x）
y 分量：z·x − x·z（跳過 y，符號翻轉因為展開時 j 前有負號）
z 分量：x·y − y·x（跳過 z）

另一種記法：寫出循環 x→y→z→x，叉積的每個分量都是「後兩個的乘積差」。

2
幾何意義：方向與大小

方向：右手定則（Right-Hand Rule）

a × b 的方向垂直於 a 和 b 所構成的平面，具體朝哪一側由右手定則決定： 伸出右手，四指從 a 的方向彎向 b 的方向，大拇指所指的方向就是 a × b。

大小：平行四邊形面積

叉積的長度（magnitude）等於以 a 和 b 為邊的平行四邊形面積：

$|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}|\,|\mathbf{b}|\sin\theta \quad (0° \le \theta \le 180°,\;\sin\theta \ge 0)$ $\begin{array}{rcl} \theta = 0° \text{ 或 } 180°\text{（平行）} & \Rightarrow & \textcolor{#f38ba8}{\mathbf{a} \times \mathbf{b} = \mathbf{0}}\\[6pt] \theta = 90°\text{（垂直）} & \Rightarrow & \textcolor{#a6e3a1}{|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}||\mathbf{b}|}\text{（面積最大）} \end{array}$
對比點積與叉積：
點積 a·b = |a||b|cosθ → 衡量「同向程度」，垂直時為 0
叉積 |a×b| = |a||b|sinθ → 衡量「垂直程度」，平行時為 0

兩者互補：cos²θ + sin²θ = 1，聯立可以同時描述向量的全部幾何關係。

3
重要性質

性質 公式 備註
反交換律 a × b = −(b × a) 順序相反，結果方向相反。這是叉積最重要的性質，與點積的交換律完全不同。
自叉積為零 a × a = 0 由反交換律推出：a×a = −(a×a) → 2(a×a) = 0 → a×a = 0
純量乘法 (ka) × b = k(a × b) 純量可以提出，與矩陣行列式的性質一致
向量加法分配律 a × (b + c) = a×b + a×c 對加法分配，可以用行列式線性性質推導
不滿足結合律 a × (b × c) ≠ (a × b) × c 叉積不可隨意加括號。展開後結果完全不同。
結果垂直於兩向量 (a × b) · a = 0
(a × b) · b = 0		 可以用分量展開直接驗證，是「法向量」用途的根據

4
與 Ch08 的 2D 叉積：同一回事

在 Ch08（Edge Function）中，我們計算過一個「2D 叉積」來判斷點在邊的哪一側：

Ch08 的 2D 叉積（回顧）：

$\text{cross2D}(\mathbf{a},\mathbf{b}) = a_x b_y - a_y b_x$

令 \(a_z = b_z = 0\)（向量在 z=0 平面上），代入 3D 叉積：

$\mathbf{a} \times \mathbf{b}\big|_{a_z=b_z=0} = \bigl(\;0,\;\;0,\;\;\textcolor{#f9e2af}{a_x b_y - a_y b_x}\;\bigr)$

z 分量就是 2D 叉積！

當兩個向量都在 xy 平面（z = 0）時，3D 叉積的 x、y 分量必然為零， 只剩 z 分量——而那個 z 分量就是 Ch08 的 2D 叉積。

這意味著 Ch08 裡所有「叉積判斷哪一側」的直覺，在 3D 裡仍然成立： z 分量的正負號就是右手定則在 z 方向的投影，對應同一側或反側。

一句話總結： 2D 叉積是 3D 叉積的 z 分量在 z=0 平面上的特例。 3D 叉積是 2D 叉積的自然延伸，把「方向」從純量擴展到向量。

5
應用：三角形法向量

光柵化時每個三角形都有三個頂點 v0, v1, v2。 從 v0 出發可以建立兩條邊向量：

$\mathbf{e}_1 = \mathbf{v}_1 - \mathbf{v}_0, \qquad \mathbf{e}_2 = \mathbf{v}_2 - \mathbf{v}_0$ $\mathbf{N} = \mathbf{e}_1 \times \mathbf{e}_2$ $\hat{N} = \dfrac{\mathbf{N}}{|\mathbf{N}|}$
e1 × e2 的方向取決於頂點繞向（winding order）
若 v₀ → v₁ → v₂ 是逆時針（CCW），N 朝向外（朝向觀察者）。
若頂點是順時針（CW），N 方向相反——可用 e2 × e1 修正，或在讀資料時保持一致的繞向。

這就是為什麼 Ch12 強調頂點繞向的一致性：繞向錯誤會讓法向量指向錯誤方向，導致光照完全不對。

6
應用：背面剔除（Back-face Culling）

計算出法向量 N 之後，可以用它來做背面剔除（back-face culling）： 判斷這個三角形是否朝向攝影機，若背對攝影機則直接跳過，省去光柵化的開銷。

設 \(\hat{V}\) 為攝影機方向（指向觀察者；通常是 \((0,0,1)\) 或 view direction 的反向）：

$\begin{array}{ll} \textcolor{#a6e3a1}{\hat{N} \cdot \hat{V} > 0} & \Rightarrow\;\text{法向量朝向攝影機}\Rightarrow\textbf{正面（front-face），需要渲染}\\[6pt] \textcolor{#f38ba8}{\hat{N} \cdot \hat{V} < 0} & \Rightarrow\;\text{法向量背向攝影機}\Rightarrow\textbf{背面（back-face），可以剔除}\\[6pt] \hat{N} \cdot \hat{V} = 0 & \Rightarrow\;\text{三角形平行於視線，邊緣情況} \end{array}$

這裡出現了 Ch13 的點積（N·V）和本章的叉積（計算 N）的組合： 兩章的工具協同工作，完成一個完整的幾何判斷。 在 MVP 管線完成前，可以先在物件空間或世界空間做背面剔除， 減少後續計算量。

7
C++ 實作

叉積的實作直接對應公式，沒有任何近似或技巧——只需確保分量順序正確：

// 延伸 Ch13 的 Vec3，加入向量加減與叉積 struct Vec3 { float x, y, z; Vec3 operator-(const Vec3& o) const { return {x-o.x, y-o.y, z-o.z}; } Vec3 operator+(const Vec3& o) const { return {x+o.x, y+o.y, z+o.z}; } Vec3 operator*(float t) const { return {x*t, y*t, z*t}; } float length_sq() const { return x*x + y*y + z*z; } float length() const { return std::sqrt(length_sq()); } Vec3 normalize() const { float l = length(); return {x/l, y/l, z/l}; } }; // 點積（Ch13） float dot(Vec3 a, Vec3 b) { return a.x*b.x + a.y*b.y + a.z*b.z; } // 叉積：結果垂直於 a 和 b Vec3 cross(Vec3 a, Vec3 b) { return { a.y * b.z - a.z * b.y, // x a.z * b.x - a.x * b.z, // y a.x * b.y - a.y * b.x // z }; } // 從三角形三頂點計算單位法向量 Vec3 triangle_normal(Vec3 v0, Vec3 v1, Vec3 v2) { Vec3 e1 = v1 - v0; Vec3 e2 = v2 - v0; return cross(e1, e2).normalize(); } // 背面剔除：V 為攝影機方向（世界空間，朝向觀察者） bool is_back_face(Vec3 v0, Vec3 v1, Vec3 v2, Vec3 view_dir) { Vec3 N = triangle_normal(v0, v1, v2); return dot(N, view_dir) < 0.0f; }
分量順序容易寫錯：叉積的 y 分量是 a.z*b.x − a.x*b.z（注意：不是 a.x*b.z − a.z*b.x）。 每個分量展開時都有一個「減號在前」的部分，y 分量整體還帶負號（行列式展開的符號規則）。 最保險的方式是對照記憶法：x→yz−zy，y→zx−xz，z→xy−yx。
